# FRUITECH

## Título do projeto

- Sistema de Irrigação autônomo

## Descrição 

O projeto visa uma melhor irrigação do solo, de forma atônoma, onde as ações são tomadas através da análise de 
umidade do solo, verificando a necessidade ou não de água. Juntamente com um sistemas de linhas independente que 
evita a perca do fluido quando houver vazamentos fechando somente a linha que está com problemas, sem prejudicar 
toda a irrigação do campo. 

## Contendo:
- Malha de controle hidraúlica autônoma;
- Irrigação do solo conforme nível de úmidade;
- Intertravamento de linhas ao diagnosticar vazamento em algum ponto, anlisando variação de pressão no sistema.

## Passo-a-passo

Faça a instalação hidráulica como tubos e conecte as valculas solenoide; 
Instale os componentes do arduino;
Após a instação dos componentes abra o script no arduino e faça a modificação das portas conforme a instalação;
Deixe os sensores de condutividade nos recipientes com terra que serão irrigados e realize os testes.

Para instalação real, faça projeto conforme a proposta no repositório
